# Configuration Guide

## Overview

> This guide provides step-by-step instructions to set up a basic server on **Ubuntu 20.04 LTS** with the following services.

- [Apache2](apache2)
- [ISC DHCP Server](isc-dhcp-server)
- [Netplan](netplan)
- [OpenSSH Server](openssh-server)
- [Samba](samba)
- [vsftpd](vsftpd.conf)
